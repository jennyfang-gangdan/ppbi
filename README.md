## 🌟 项目简介

**PawPaw BI** 是一款基于 React 开发的趣味性格测试 H5 应用。它模仿经典的 MBTI 理论，从 **社交 (E/I)**、**动力 (F/N)**、**情感 (A/T)**、**纪律 (J/P)** 四个维度对狗狗的行为进行分析，最终生成一份包含性格称号、MBTI 代码、灵魂伴侣及匹配铲屎官的“绝密狗格报告”。

### 🐶 四大鉴定维度
| 维度 | 属性 A | 属性 B |
| :--- | :--- | :--- |
| **社交 (Energy)** | **E** (社交悍匪) | **I** (社恐宅狗) |
| **动力 (Motivation)** | **F** (干饭狂魔) | **N** (脑洞探险家) |
| **情感 (Affection)** | **A** (极度粘人) | **T** (高冷霸总) |
| **纪律 (Discipline)** | **J** (守规矩) | **P** (拆家狂魔) |

---

## ✨ 功能亮点

- **🎨 还原级 UI 设计**：深度参考原生 App 视觉，包含跳动的狗头动画、可爱的狗爪选项图标。
- **📊 科学算法**：内置 15 道精选题目，通过加权计算得出最接近的性格画像。
- **🖼️ 结果卡片生成**：支持生成包含 Emoji、标签、详细描述及“匹配铲屎官”建议的精美结果页。
- **🚀 零环境运行**：单 HTML 文件架构，通过 CDN 引入 React 和 Tailwind CSS，无需安装 Node.js 即可运行。
- **📱 响应式适配**：完美适配手机端浏览器，支持长按截图保存分享。

## 🛠️ 技术栈

- **Core**: [React 18](https://reactjs.org/) (via CDN)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Logic**: Babel (In-browser JSX transformation)
- **Design Elements**: SVG Icons & CSS3 Animations

## 🚀 快速开始

### 方式一：本地运行
1. 下载本项目中的 `index.html` 文件。
2. 直接使用 Chrome、Edge 或 Safari 浏览器打开该文件。

### 方式二：GitHub Pages 部署 (推荐)
1. 将代码上传到你的 GitHub 仓库。
2. 进入仓库的 `Settings` > `Pages`。
3. 在 `Build and deployment` 下选择 `Deploy from a branch`。
4. 刷新后即可获得一个在线访问链接。
